## React-App-Main
- React 17.0.2
- webpack 5.48.0
* UI Ant Design

## Installation

```
window 10
nodejs 14.x.x
git
vscode
```

`git clone https://github.com/kdg1986/react-app-main.git`

`cd react-app-main`

`npm i -g yarn`

`yarn`

`yarn start`

## Start

`yarn start`

`http://localhost:3000`

## Build

`yarn build -> ./build`

`yarn build-ele -> ./dist`
