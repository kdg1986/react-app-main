## WDMS

- React 17.0.2
- webpack 5.48.0
- Ant Design 4.16.13
- electron 13.2.1

## Installation

```
window 10
nodejs 14.x.x
git
vscode
```

`git clone http://172.16.30.31/webFe/WDMS.git`

`cd WDMS`

`npm i -g yarn(npm으로 해도됨..)`

`yarn`

`yarn start`

## Start

`yarn start`

`http://localhost:3000`

## Build

`yarn build -> ./build`

`yarn build-ele -> ./dist`
